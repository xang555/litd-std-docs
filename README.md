# Company Development Standards

This repository contains the official development standards, guidelines, and shared resources for all engineering teams.

## Structure

- **`teams/`** - Team-specific standards and templates
- **`skills/`** - Shared skills and learning resources
- **`shared/`** - Cross-team standards and conventions
- **`agents/`** - AI agent instructions and prompts

## Accessing Standards

Standards are accessible through the [LitD MCP Server](https://github.com/company/litd-rmcp).

## Contributing

1. All changes must go through pull request
2. Get approval from the Standards Team
3. Reference relevant issue or RFC
4. Update related documentation

## Version Control

- Main branch reflects current standards
- Use semantic versioning for major updates
- Document breaking changes in migration guides

## Contact

For questions or suggestions, contact the Engineering Standards Team.
