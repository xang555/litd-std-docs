# Problem Solving

## Description

Core problem-solving skills and methodologies for effective troubleshooting and debugging.

## Level: Junior

### Learning Objectives

- Understand systematic debugging approaches
- Learn to read error messages effectively
- Practice rubber duck debugging

### Required Skills

1. **Error Analysis**

   - Read stack traces from bottom to top
   - Identify error types and their meanings
   - Locate the exact line causing the error

2. **Reproduction Steps**

   - Write down steps to reproduce bugs
   - Identify minimal reproduction cases
   - Use console.log strategically

3. **Documentation Usage**

   - Navigate official documentation
   - Search effectively
   - Find relevant examples

### Resources

- [Debugging Guide](https://example.com/debugging)
- Company troubleshooting wiki

## Level: Mid-Level

### Learning Objectives

- Use debugging tools effectively
- Understand common anti-patterns
- Practice root cause analysis

### Required Skills

1. **Tool Proficiency**

   - Browser DevTools (breakpoints, stepping)
   - Debugger statements
   - Performance profiling

2. **Root Cause Analysis**

   - Five Whys technique
   - Fishbone diagrams
   - Timeline analysis

3. **Testing Hypotheses**

   - Form testable hypotheses
   - Design experiments
   - Document findings

### Optional Skills

- Memory profiling
- Network analysis
- A/B testing for bug verification

## Level: Senior

### Learning Objectives

- Mentor others in debugging
- Design debuggable systems
- Lead incident response

### Required Skills

1. **System Design**

   - Design for observability
   - Implement structured logging
   - Create debugging documentation

2. **Incident Management**

   - Post-mortem writing
   - Blameless culture
   - Prevention strategies

3. **Knowledge Sharing**

   - Debugging sessions
   - Case studies
   - Training materials
