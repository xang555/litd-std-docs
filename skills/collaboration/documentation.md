# Documentation

## Description

Writing clear, useful documentation for code and systems.

## Level: Junior

### Learning Objectives

- Document code effectively
- Write clear commit messages
- Contribute to project docs

### Required Skills

1. **Code Documentation**

   - Function/class comments
   - Inline comments
   - When to document

2. **Commit Messages**

   - Conventional commits
   - Clear descriptions
   - Reference issues

3. **README Basics**

   - Project overview
   - Setup instructions
   - Usage examples

### Resources

- Documentation style guide
- Technical writing tips

## Level: Mid-Level

### Learning Objectives

- Write comprehensive docs
- Document APIs
- Maintain documentation

### Required Skills

1. **API Documentation**

   - Endpoint documentation
   - Type definitions
   - Examples and use cases

2. **Architecture Docs**

   - System diagrams
   - Design decisions
   - Architecture decision records (ADRs)

3. **Maintenance**

   - Keeping docs current
   - Documentation reviews
   - Versioning docs

### Optional Skills

- Interactive tutorials
- Video documentation
- Documentation tools

## Level: Senior

### Learning Objectives

- Establish documentation standards
- Build documentation culture
- Create doc infrastructure

### Required Skills

1. **Documentation Strategy**

   - Doc types and purposes
   - Audience analysis
   - Delivery formats

2. **Infrastructure**

   - Documentation platforms
   - Automated docs
   - Searchability

3. **Culture Building**

   - Doc as code
   - Documentation first
   - Recognition practices
