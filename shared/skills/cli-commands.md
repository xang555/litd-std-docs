# CLI Commands

## Description

Mastering command-line tools for development efficiency.

## Level: Junior

### Learning Objectives

- Navigate the shell
- Use common commands
- Understand file operations

### Required Skills

1. **Navigation**

   - ls, cd, pwd
   - Tab completion
   - History

2. **File Operations**

   - cat, less, head, tail
   - cp, mv, rm
   - mkdir, touch

3. **Common Tools**

   - grep, find
   - ps, kill
   - curl, wget

### Resources

- Shell reference guide

## Level: Mid-Level

### Learning Objectives

- Use advanced shell features
- Create command aliases
- Script common tasks

### Required Skills

1. **Shell Features**

   - Pipes and redirection
   - Command substitution
   - Background jobs

2. **Productivity**

   - Aliases and functions
   - Shell configuration
   - Custom prompts

3. **Advanced Tools**

   - awk, sed
   - xargs
   - jq for JSON

### Optional Skills

- Shell scripting
- Advanced regex
- System administration
