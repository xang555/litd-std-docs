# Communication

## Description

Effective communication strategies for engineering teams.

## Level: Junior

### Learning Objectives

- Communicate clearly in writing
- Participate in discussions
- Ask effective questions

### Required Skills

1. **Written Communication**

   - Clear emails/messages
   - Appropriate channels
   - Professional tone

2. **Asking Questions**

   - Provide context
   - Show research effort
   - Be specific

3. **Participation**

   - Standup updates
   - Meeting etiquette
   - Active listening

### Resources

- Communication guidelines
- Team charter

## Level: Mid-Level

### Learning Objectives

- Facilitate discussions
- Present technical information
- Give and receive feedback

### Required Skills

1. **Facilitation**

   - Running meetings
   - Managing discussions
   - Consensus building

2. **Presentations**

   - Technical presentations
   - Demo preparation
   - Q&A handling

3. **Feedback**

   - Giving constructive feedback
   - Receiving feedback
   - Performance discussions

### Optional Skills

- Conflict resolution
- Negotiation skills
- Cross-team communication

## Level: Senior

### Learning Objectives

- Lead communication strategy
- Build communication culture
- Handle sensitive situations

### Required Skills

1. **Strategic Communication**

   - Stakeholder management
   - Executive updates
   - Cross-functional alignment

2. **Culture Building**

   - Psychological safety
   - Inclusive communication
   - Knowledge sharing

3. **Challenging Situations**

   - Delivering bad news
   - Handling conflicts
   - Crisis communication
