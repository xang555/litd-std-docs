# Technical Writing

## Description
Writing clear, concise, and effective technical documentation.

## Level: Junior

### Learning Objectives
- Understand different types of technical documentation
- Learn basic writing principles (clarity, conciseness, completeness)
- Write effective commit messages
- Document code with comments and docstrings

### Required Skills
- Basic writing skills
- Understanding of code structure

### Resources
- [Google Technical Writing One](https://developers.google.com/tech-writing/one)
- [Documentation Style Guides](https://www.writethedocs.org/)

## Level: Mid-Level

### Learning Objectives
- Write comprehensive API documentation
- Create effective architecture diagrams
- Write runbooks and troubleshooting guides
- Document complex algorithms
- Review and improve others' documentation

### Required Skills
- Junior technical writing skills
- Understanding of software architecture
- Experience with documentation tools (Markdown, OpenAPI, etc.)

### Optional Skills
- Diagramming tools (Mermaid, PlantUML)
- Documentation site generators (Docusaurus, MkDocs)
- Video tutorials and screencasts

## Level: Senior

### Learning Objectives
- Design documentation strategy for organizations
- Create and maintain style guides
- Mentor others in technical writing
- Balance documentation with development velocity
- Measure documentation effectiveness

### Required Skills
- Mid-level technical writing skills
- Experience with documentation at scale
- Understanding of information architecture
- Leadership and mentoring experience
