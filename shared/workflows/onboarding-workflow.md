# Team Member Onboarding Workflow

## Description
A structured workflow for onboarding new team members effectively.

## Prerequisites
- New hire start date confirmed
- Equipment and access provisioned
- Manager assigned
- Desk/workspace assigned

## Steps

### 1. Pre-arrival Preparation (1 week before)
- **Owner**: HR + Manager
- **Tasks**:
  - Send welcome email with first-day information
  - Prepare equipment (laptop, peripherals)
  - Set up email and accounts
  - Assign onboarding buddy
- **Output**: New hire welcome packet sent

### 2. First Day - Orientation
- **Owner**: Manager + Buddy
- **Tasks**:
  - Office tour and introduction
  - Set up workstation and tools
  - Review org chart and team structure
  - Set up 1:1 with manager
  - Assign initial tasks
- **Output**: New hire setup and oriented

### 3. First Week - Learning
- **Owner**: Buddy + Manager
- **Tasks**:
  - Review team documentation
  - Clone and build project locally
  - Walk through architecture
  - Introduce to key stakeholders
  - Set up development environment
  - Complete first small task/bug fix
- **Output**: Development environment working, first task completed

### 4. First Month - Integration
- **Owner**: Manager
- **Tasks**:
  - Weekly 1:1 check-ins
  - Attend sprint ceremonies
  - Work on assigned tickets
  - Present in team meeting
  - Review team standards and practices
  - Shadow team members on tasks
- **Output**: Contributing to team goals

### 5. First Quarter - Full Contribution
- **Owner**: Manager
- **Tasks**:
  - Bi-weekly 1:1 check-ins
  - Handle features independently
  - Participate in code reviews
  - Lead small initiatives
  - Provide feedback on onboarding
  - Set quarterly goals
- **Output**: Fully integrated team member

## Outputs

| Output | Description | Format |
|--------|-------------|--------|
| Onboarding checklist | Task tracker | Document |
| Access request form | System access | Form |
| 30-60-90 day plan | Goals and expectations | Document |
| Feedback survey | Onboarding feedback | Survey |

## Success Criteria

- New hire completes first task within first week
- New hire contributes to sprint within first month
- Positive feedback from new hire and team
- Retention at 90 days

## Owner

**Primary**: Engineering Manager
**Support**: Team Lead, Onboarding Buddy
