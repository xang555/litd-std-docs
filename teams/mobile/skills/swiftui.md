# SwiftUI

## Description
Building native iOS user interfaces using SwiftUI's declarative syntax.

## Level: Junior

### Learning Objectives
- Understand SwiftUI's declarative paradigm
- Learn basic views: Text, Image, VStack, HStack, Button
- Implement state with @State
- Build simple forms and lists

### Required Skills
- Basic Swift knowledge
- Understanding of iOS app lifecycle
- Xcode basics

### Resources
- [Apple SwiftUI Documentation](https://developer.apple.com/documentation/swiftui)
- [SwiftUI Tutorials](https://developer.apple.com/tutorials/swiftui)

## Level: Mid-Level

### Learning Objectives
- Master @Binding, @ObservedObject, @StateObject
- Create custom ViewModifiers
- Implement animations and transitions
- Handle gestures
- Combine with UIKit when needed

### Required Skills
- Junior SwiftUI skills
- Understanding of data flow
- Basic Combine framework knowledge

### Optional Skills
- SwiftUI Previews
- Custom shapes and paths
- Accessibility in SwiftUI

## Level: Senior

### Learning Objectives
- Design complex SwiftUI architectures
- Create reusable component libraries
- Performance optimization for complex views
- Advanced animations and gestures
- Teaching SwiftUI best practices

### Required Skills
- Mid-level SwiftUI skills
- Deep understanding of iOS UI patterns
- Performance profiling experience
