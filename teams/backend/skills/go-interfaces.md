# Go Interfaces

## Description
Understanding and effectively using Go interfaces for abstraction and polymorphism.

## Level: Junior

### Learning Objectives
- Understand what interfaces are in Go
- Learn about implicit interface implementation
- Use common interfaces from standard library (io.Reader, io.Writer)

### Required Skills
- Basic Go syntax and types
- Understanding of functions and methods

### Resources
- [Go Interfaces Documentation](https://go.dev/doc/effective_go#interfaces)
- [Interfaces in Go](https://go.dev/tour/methods/9)

## Level: Mid-Level

### Learning Objectives
- Design custom interfaces for your packages
- Understand interface composition
- Use interfaces for dependency injection
- Apply the "accept interfaces, return structs" principle
- Handle interface nil values correctly

### Required Skills
- Junior Go Interface skills
- Understanding of pointers and receivers
- Package design principles

### Optional Skills
- Code generation with interfaces
- Mocking interfaces for testing

## Level: Senior

### Learning Objectives
- Design interface hierarchies that remain stable over time
- Understand interface pollution and how to avoid it
- Use interfaces to enable testing without over-abstraction
- Design APIs that are easy to mock and extend
- Teach interface design best practices to team members

### Required Skills
- Mid-level Go Interface skills
- Experience with large codebase maintenance
- Understanding of API versioning and compatibility
